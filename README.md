# analizator
Выполнить следующие задания для одного из вариантов конструкций.
1.	Построить синтаксические диаграммы дтя конструкции в соответствии с вариантом задания.
2.	Запрограммировать синтаксический анализатор методом рекурсивного спуска. Анализатор должен либо сообщать о том: что конструкция записана верно, либо выдавать сообщение об ошибке с указанием места ее обнаружения.
3.	Дополнить анализаторы семантическими процедурами, выполняющими содержательную обработку.
4.	Подготовить тестовые примеры входных строк для программ в текстовых файлах.

Последовательность записанных через точку с запятой описаний двумерных массивов без инициализирующих выражений то есть описаний вида имя |размерность1][размерность2]. Где имя - последовательно латинских букв и цифр, начинающаяся не с цифры, размерность — целое число>0 и <1000. Количество описаний не менее одного. Семантическая процедура - сгенерировать программу на С++, случайным образом заполняющую эти массивы и выводящую их на экран.

Дополнительно была реализована проверка на повторную декларацию массива с тем же именем и объявление массивов с нулевым размером.

Код делался попроще, чтоб соответствовать уровню студента.
